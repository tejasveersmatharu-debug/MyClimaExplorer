# MyClimaExplorer
Hi!
</br>
This is MyClimaExplorer, a project made by Four First-year students of IIT-BHU in Hackathon-Hack it Out.
</br>
This project was made independently under 24-hrs, and contains many features, 
<br>
such as showing GRAPHS, HEATMAPS, 3D GLOBES, and even shows COMPARISONS!
</br>
It Has a vibrant looking UI, made with HTML, CSS and Javascript. The Backend is made using foundation as Python.
</br>
In python, it uses many libraries such as Flask, Pandas, Plotly, Numpy, Xarray, and Numpy.express.
</br>
It i scurrently locally hoisted.
</br>
This stack helps us keep the vibrant UI, along with functionality. Javascript makes it look like a proper Website, </br> which is 
not the same in case of tools like streamlit.
It also makes our Website much more functional and Interactive.
</br>
